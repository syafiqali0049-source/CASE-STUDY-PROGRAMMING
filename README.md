# 🌍 Sistem Log Digital Makmal Geomatik

Sistem berasaskan web untuk menggantikan buku log fizikal di makmal komputer Geomatik.

## Ciri-ciri:
- Antara muka mesra peranti mudah alih (Bootstrap 5).
- Pangkalan data automatik (SQLite).
- Rekod masa penggunaan secara real-time.

## Cara Menjalankan Projek:
1. Pasang Python.
2. Pasang library: `pip install -r requirements.txt`.
3. Jalankan aplikasi: `python app.py`.
4. Buka di pelayar: `http://127.0.0.1:5000`.
